static oGetCommonTicketPathForFullPath (line 42)

Returns the common path all paths on the given path will share

Example: If the full path is 

/branches/my-hammer2/_production/2010-04-15/cooperationsbox/log_conrad_123x30_AT.gif
and one of the source paths is 
/branches/my-hammer2/_production
then the common ticket path is 
/branches/my-hammer2/_production/2010-04-15

  • return: MergeHelper_RepoPath if a common path could be found, NULL if none of the $oRepo source paths matched the given path
